### Установка git на ваш ПК
Для установки git на Windows, необзодимо скачать пакет установки с сайта [gitforwindows.org](https://gitforwindows.org)

#### Первая настройка git
Для первоначальной настройки необходимо представиться гиту следующими командами:
```
git config --global user.name "John Doe"
git config --global user.email johndoe@example.com
```
