### Базовая работа с ветками в git.

1. Создание новой ветки команда git branch "NameBranch"

Либо git checkout -b NameBranch (для содания и перехода сразу в новую ветку)

2. git branch -d имя ветки (удаление нужной СЛИТОЙ ветки)

3. git branch -D имя ветки (удаление ветки даже не сохраненной АККУРАТНЕЕ с данной командой)
