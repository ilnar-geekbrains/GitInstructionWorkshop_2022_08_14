## Ветки. Локальная работа с ветками в git.

### Что такое ветки и для чего они нужны при работе с системой контроля версий.

### Базовая работа с ветками в git. - для создания ветки нужно ввести git branch branch_name. Для перемещения по веткам используйте git checkout branch_name. Git branch выводит список всех не скрытых веток. Чтобы удалить слитую ветку git branch -d branch_name, чтобы удалить ветку в принципе: git branch -D branch_name.
